# 📚 Neo E-Library (Appperpustakaan)

Aplikasi perpustakaan berbasis web menggunakan **Laravel 12** yang menyediakan fitur katalog buku, keranjang peminjaman, peminjaman online, serta panel admin untuk verifikasi dan pengelolaan data.

---

## 🚀 Fitur Utama
- 📖 Katalog buku
- 🛒 Keranjang peminjaman
- 📥 Peminjaman buku secara online
- 👨‍💼 Panel admin (approve / verifikasi)
- 👨‍🎓 Registrasi & login mahasiswa
- 📊 Riwayat peminjaman

---

## ⚙️ Kebutuhan Sistem
- Windows 10/11  
- XAMPP (PHP 8.2 atau lebih baru)  
- Composer  
- MySQL / MariaDB  
- (Opsional) Node.js & NPM
